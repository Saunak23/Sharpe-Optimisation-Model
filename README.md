# Sharpe-Optimisation-Model
Select 10 sectors, and from each sector study 10 companies (total 100 companies). Based on the fundamental analysis of these 100 companies (only quantitative), select one sector from each sector for further sharpe optimization model

Excel Structure (3 Sheets):

Sheet 1 - Company/Stock Analysis of 100 companies from 10 sectors.

Sheet 2 - Sharpe Optimization Model, we have downloaded price data of last one year

Sheet 3 - Portfolio Risk & Return Calculation, calculate utility by dividing portfolio return by portfolio risk
